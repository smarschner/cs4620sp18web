CS 4621: Lecture 4

Interaction

  • For many applications, we would like for the users to be able to interact with elements displayed on the webpage.
    • For example, through keystrokes and mouse buttons.
  • Fortunately, browsers provide an event system to capture these events.
  • JavaScript callback functions can be run on user input, and in turn, this can change WebGL behavior.
